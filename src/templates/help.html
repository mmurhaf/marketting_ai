📸 Instagram Configuration

Important: Instagram has two integration methods. The official API is recommended for business accounts, while the unofficial method works for personal accounts but carries more risk.

Method 1: Instagram Graph API OFFICIAL RECOMMENDED

Requirements:

  • Instagram Business or Creator account
  • Facebook Page connected to your Instagram account
  • Facebook Developer account

Step-by-Step Instructions:

1 Convert to Business Account
  • Open Instagram app → Go to Profile
  • Tap menu (☰) → Settings → Account
  • Tap "Switch to Professional Account"
  • Choose "Business" or "Creator"
2 Connect to Facebook Page
  • In Instagram settings → Business → Linked Accounts
  • Tap Facebook and connect your page
  • If you don't have a page, create one at facebook.com/pages/create
3 Create Facebook App
  • Go to Meta for Developers > My Apps
  • Click "Create App" → Select "Business" type
  • Fill in app details and create.
4 Add Products
  • In your app dashboard, find "Instagram Graph API" and click "Set Up".
  • (Optional) Add "Facebook Login for Business" if asked, but Graph API is the key one.
5 Generate Access Token
  • Go to Graph API Explorer
  • Select your App from the "Meta App" dropdown.
  • Crucial: Change "User or Page" dropdown to "Get Page Access Token".
  • Select your Facebook Page.
  • Add permissions: instagram_basic, instagram_content_publish, pages_read_engagement, pages_manage_posts
  • Click "Generate Access Token".
6 Get Instagram Business Account ID
  • In the Graph API Explorer query box, enter: me?fields=instagram_business_account
  • Click "Submit".
  • Copy the ID from the response (inside instagram_business_account: { id: "..." }).
7 Extend Token (Recommended)
  • Click the "i" info icon next to your token in the Explorer.
  • Click "Open in Access Token Tool".
  • Click "Extend Access Token" to get a 60-day token.
✅ Paste the Access Token in the "Facebook Access Token" field.
✅ Paste the Business Account ID in the "Instagram Business Account ID" field.

Method 2: Username & Password UNOFFICIAL

⚠️ Warning: This method uses unofficial APIs and may violate Instagram's Terms of Service. Your account could be banned or suspended. Use at your own risk!

This method requires installing additional libraries like instagrapi:

pip install instagrapi

To use this method:

  1. Enter your Instagram username in the Instagram Username field
  2. Enter your Instagram password in the Instagram Password field
  3. Save configuration
Note: This method is currently stored but not implemented in the code. If you want to use it, the developer needs to integrate the instagrapi library into the publisher module.

👍 Facebook Configuration

Facebook Page Access Token OFFICIAL

Requirements:

  • Facebook Page (not personal profile)
  • Facebook Developer account
  • Page admin access
Tip: If you already set up the Instagram Graph API (Method 1) above, you can use the same Access Token! Just ensure you added the publish_video permission to it. You don't need a separate token.
1 Create or Access Your Facebook Page
  • Go to facebook.com/pages
  • Create a new page or select an existing one
  • Ensure you are an admin of the page
2 Create Facebook App (if not already done)
  • Go to developers.facebook.com
  • Click "My Apps" → "Create App"
  • Choose "Business" type
  • Fill in details and create
3 Add Required Products
  • In app dashboard, add "Facebook Login"
  • Add "Graph API" if not already added
4 Generate Page Access Token
  • Go to Tools → Graph API Explorer
  • Select your app from the dropdown
  • Click "Get Token" → "Get Page Access Token"
  • Select your page and authorize
  • Add permissions: pages_manage_posts, pages_read_engagement, publish_video
  • Copy the generated token
5 Make Token Long-Lived (Recommended)
  • Go to Access Token Debugger
  • Paste your token and click "Debug"
  • Click "Extend Access Token" to get a 60-day token
  • Copy the new extended token
✅ Paste the Page Access Token in the Facebook Access Token field on the Settings page.

💡 Tip: After saving, use the "Test token" button to verify it works!
Token Permissions Required:
  • pages_manage_posts - Publish posts to your page
  • pages_read_engagement - Read engagement metrics
  • publish_video - Upload video content

🎵 TikTok Configuration

Session ID Method UNOFFICIAL

⚠️ Warning: This is an unofficial method that extracts your session cookie. It may stop working if TikTok changes their system, and could potentially violate their Terms of Service.

How to Get Your TikTok Session ID:

1 Log in to TikTok on Desktop Browser
  • Open Chrome, Edge, or Firefox
  • Go to www.tiktok.com
  • Log in to your account
2 Open Developer Tools
  • Right-click anywhere on the page
  • Click "Inspect" or press F12
  • Developer tools will open at the bottom or side
3 Navigate to Application/Storage Tab
  • In developer tools, click the "Application" tab (Chrome/Edge)
  • Or "Storage" tab (Firefox)
4 Find Session Cookie
  • In the left sidebar, expand "Cookies"
  • Click on https://www.tiktok.com
  • Scroll through the list and find sessionid
  • Click on it to view the value
5 Copy the Session ID
  • Double-click the Value field
  • Copy the entire string (usually long and alphanumeric)
  • It will look something like: 3a4b5c6d7e8f9g0h1i2j3k4l5m6n7o8p
✅ Paste the Session ID in the TikTok Session ID field on the Settings page.
Session Expiry: Session IDs can expire after a period of inactivity or if you log out. If publishing stops working, you may need to obtain a fresh session ID.

TikTok Official API OFFICIAL COMPLEX

TikTok offers an official Content Posting API, but it requires:

  • Approval from TikTok (application process)
  • Business/Developer account
  • OAuth authentication flow
  • Webhook setup for notifications

Learn more: TikTok for Developers

This application currently doesn't support the official TikTok API due to its complexity. Consider using the Session ID method for now.

🧪 Testing Your Configuration

1 Save Your Credentials

Go to the Settings page and enter all your credentials, then click "Save Configuration".

2 Test Facebook Token

On the Settings page, click the "Test token" button next to the Facebook Access Token field. You should see "OK" if the token is valid.

3 Run a Test Scrape

Go to the Dashboard and try scraping some images to test the system.

4 Create Test Content

Use the Editor to create a test video reel from your scraped images.

5 Test Publishing

Attempt to publish the test video.

Note: The current system runs in Simulation Mode. It will validate that your configuration is loaded, but will not actually upload content to social media platforms yet. Check the server logs (or the logs page) to see the success message.

🔧 Troubleshooting

Common Issues

Token Invalid or Expired

  • Facebook/Instagram tokens expire. Generate a new long-lived token.
  • Use the Access Token Debugger to check expiration dates.

Insufficient Permissions

  • When generating tokens, ensure you select all required permissions.
  • You may need to re-authorize the app with additional permissions.

Instagram Not Connecting to Facebook Page

  • Make sure your Instagram account is converted to Business/Creator.
  • Verify the page connection in Instagram settings.
  • You must be an admin of the Facebook Page.

TikTok Session ID Not Working

  • Make sure you copied the entire session ID value.
  • Try logging out and back in to TikTok, then get a fresh session ID.
  • Check that you're copying from www.tiktok.com, not a country-specific domain.

Publishing Errors

  • Verify your token/credentials are still valid and saved in Settings.
  • Check the Server Logs for specific error details.
